KI Sushi - Style Guide

LOGO

1. Logo should have at least 8px on of space on all sides

2. Logo should only appear on a page with either a solid white or dark background (a dark texture maybe use in the background)

COLORS

hex: #000

hex: #fff

hex: #931b1f

hex: #1d9e10

1. Only colors above should be use

FONTS

All fonts should be Proxima Nova which can be generated by using ADOBE fonts

fonts should also have a fall back of Helvetica

Example:

font-family:'proxima-nova', Helvetica, sans-serif;

Sample of h1

This is a h1 size

h1{
font-family:'proxima-nova',Helvetica, sans-serif;
font-size:36px;
font-weight: 100;
line-height:1.4;
}
